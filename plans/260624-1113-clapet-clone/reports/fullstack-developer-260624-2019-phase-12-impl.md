## 2. Per-Format Mapping Summary

### Docker Compose
| Variant | Service |
|---|---|
| database/postgres | postgres:{version}, env, volume, port 5432 |
| database/mysql | mysql:{version}, env, volume, port 3306 |
| database/mongodb | mongo:{version}, env, volume, port 27017 |
| database/cassandra | cassandra:4, port 9042 |
| cache/redis | redis:7-alpine, --maxmemory from config |
| cache/memcached | memcached:alpine, -m from config |
| queue/rabbitmq | rabbitmq:3-management, ports 5672+15672 |
| queue/kafka | cp-kafka:7.5.0 + auto-added zookeeper service |
| storage/local-disk | busybox volume mount |
| load_balancer/nginx | nginx:alpine, volume nginx.conf |
| load_balancer/haproxy | haproxy:alpine, volume haproxy.cfg |
| worker/nodejs | node:{runtime}-alpine, working_dir, volumes |
| worker/python | python:{runtime}-slim |
| worker/go | golang:1.22-alpine |
| worker/rust | rust:1-alpine |
| api/* | your-org/{name}:latest + build, port 3000/4000/50051 |
| cdn/*, external/*, user/* | comment placeholder |
| edges → depends_on | target.depends_on includes sanitized source label |

### Terraform AWS
| Variant | Resource |
|---|---|
| database/postgres, mysql | aws_db_instance (engine, version, instance_class from connectionPool) |
| database/dynamodb | aws_dynamodb_table (billing_mode from config) |
| cache/redis, keydb, valkey | aws_elasticache_cluster engine=redis, node_type from memoryGb |
| cache/memcached | aws_elasticache_cluster engine=memcached |
| queue/sqs | aws_sqs_queue (fifo support) |
| queue/kafka | aws_msk_cluster (broker count from replicationFactor) |
| storage/s3 | aws_s3_bucket |
| storage/r2 | aws_s3_bucket + comment about provider swap |
| load_balancer/aws-alb | aws_lb + aws_lb_target_group + aws_lb_listener |
| cdn/cloudfront | aws_cloudfront_distribution (price_class) |
| worker/aws-lambda | aws_lambda_function (memory_size, timeout, architecture) |
| api/api-gateway | aws_api_gateway_rest_api |
| external/*, user/* | comment only |

### Kubernetes
| Variant | Resources |
|---|---|
| api/*, worker/* | Deployment + ClusterIP Service |
| database/postgres, mysql, mongodb, cassandra | StatefulSet + headless Service + volumeClaimTemplate |
| cache/redis, keydb, valkey | StatefulSet + headless Service |
| cache/memcached | Deployment + Service |
| queue/rabbitmq, kafka | StatefulSet + headless Service |
| storage/local-disk | PersistentVolumeClaim |
| load_balancer/nginx, haproxy | Deployment + LoadBalancer Service |
| cdn/*, external/*, user/* | comment only |
| All | namespace: archlet, label app.kubernetes.io/managed-by: archlet |

### Ansible
- Uses `community.docker.docker_container` tasks
- Nodes topologically sorted (sources before targets)
- Kafka auto-adds zookeeper task before it
- Variables extracted to `vars:` block with vault hints
- Non-containerizable variants (sqs, lambda, CDN) → comment tasks

### Raw JSON
- `{ archletVersion: "1.0", generatedAt: ISO timestamp, nodes, edges }`
- Pretty-printed, 2-space indent
